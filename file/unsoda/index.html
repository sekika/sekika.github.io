UNSODA data

This directory contains data converted from UNSODA 2.0: Unsaturated Soil Hydraulic Database. Here is how the files were obtained.

  1. UNSODA 2.0 zip package unsoda.ZIP was downloaded where the timestamp was shown as Jun 08, 2020 and the MD5 sum was shown as 7c6170d47fe8b3df3b79d2f7d56fdc02.
  2. The zip archive was unpacked and unsoda.mdb was obtained. Then it was converted to a SQLite file unsoda.sqlite (1.6 MB) with a converter at rebasedata.com.
  3. All the tables in the database were converted to json files such as general.json with this shell script. A list of tables was also produced.
  4. "Sandy laom" was manually corrected to "Sandy loam" for code 3322 in general.json.
  5. Finally, the json files were concatenated into a single json file unsoda.json (2.2 MB) with this Python script for the use from the UNSODA viewer. Data types were modified for convenience, e.g, making the code as a dictionary key.
